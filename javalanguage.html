  6. variables

    7. there are rules for naming the variable in java :
    • It can start a name for a variable with a letter , $ ,or _ , otherwise, it is not allowed
    • It can contain letters , $ , digits,or _ ,otherwise, it is not allowed
    • The names are case-sensitive, meaning that Apple and apple are two different variables.
    • There are 51 reserved words in the Java programming language. You cannot name the variables by one of these words. Reserved words are words that have a prior meaning in the language. You can visit this site to review reserved words in the JavaList of Reserved Words
