  4. Python Variables

    5. Unlike other programming languages, Python has no command for declaring a variable.

    • A variable is created the moment you first assign a value to it.
    • A variables do not need to be declared with any particular type, and can even change type after they have been set.

  5. Python Data Types

    6. In Python, the data type is set when you assign a value to a variable:

    Setting the Data Type
    Type Example Description
    int x = 20 Whole numbers
    float x = 20.50 Numbers with a decimal point
    str x = "Hello World" Ordered sequence of characters
    list x = ["apple", "banana", "cherry"] Ordered sequence of objects
    set x = {"apple", "banana", "cherry"} Unordered collection of unique objects
    bool x = True Logical value indicating True or False
